SBB API Web App

A friend was describing a simple web app he wanted that would show him the next trains departing from his local railway station, or the station nearest his office, so that he could see at a glance which were the next departures, and which were delayed or cancelled.

I decided to find out whether an API existed for the Swiss Federal Railways (SBB), and if so, whether I could use it to build the app. Via prompts with Claude, I built that app for my friend in a day, and he uses it every day on his mobile phone.

This demonstrates my ability to think creativiely to solve a real-world problem, even if loosely described, with little prior knowledge of the technical domain, but an inkling that a solution must be possible via existing tools.
