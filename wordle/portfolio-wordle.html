Wordle Simulator

I built a fully functional Wordle simulator in Google Sheets and Apps Script, but it cannot be embedded directly into my portfolio, so I reimplemented it as a web application.

It uses a custom architecture with Google Apps Script as a serverless backend API and Google Sheets as its master dictionary database — a clean separation of concerns between game logic, data, and presentation.

Unlike the real game, you can play as many times per day as you like.
